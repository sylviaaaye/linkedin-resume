# AI PM求职自动化系统 - 所需技能调研

## 🎯 项目概述
**目标**：为F1 OPT国际学生打造完整的AI Product Manager求职自动化系统
**核心功能**：LinkedIn职位监控 → 智能筛选 → 简历个性化 → 自动化申请

## 🔍 基于现有代码库的技能分析

### **现有技术资产**
1. **表单填充引擎** (`resume-autofill-extension/content/content.js`)
   - 成熟的DOM操作和表单填充逻辑
   - React Fiber支持（Ant Design, Element UI, Arco Design）
   - 元素查找系统（data-test-id, aria-label, xpath）
   - 事件触发和状态管理

2. **数据转换器** (`resume-autofill-extension/content/converters.js`)
   - 日期格式转换（YYYY-MM → MM/DD/YYYY）
   - 学位名称映射（Master → 硕士）
   - 城市信息转换（成都 → 028/CD/chengdu）
   - 智能格式推测

3. **LinkedIn监控基础** (`us-resume-personal-assistant/content/linkedin-monitor.js`)
   - DOM变化监听（MutationObserver）
   - 职位卡片解析
   - 关键词匹配算法
   - 实时评分系统

4. **Claude API集成** (`us-resume-personal-assistant/background/service-worker.js`)
   - API调用封装
   - 错误处理和重试
   - 数据解析和格式化

## 📋 所需技能分类

### **1. LinkedIn爬虫/监控技能**

#### **核心技能**
- **DOM操作和解析**
  - 熟练使用`document.querySelector`, `getElementById`, `xpath`
  - 理解CSS选择器和DOM遍历
  - 处理动态加载内容（Infinite Scroll）

- **MutationObserver API**
  - 监听DOM变化，实时检测新职位
  - 性能优化（debounce/throttle）
  - 避免无限循环检测

- **反爬虫策略**
  - 请求频率控制（避免被封IP）
  - 用户代理轮换
  - 模拟人类行为（随机延迟，鼠标移动）
  - 处理验证码和登录状态

- **LinkedIn页面结构分析**
  - 职位卡片选择器（`.job-card-container`, `[data-job-id]`）
  - 职位详情页面解析
  - 分页和筛选器交互
  - 登录状态维护

#### **进阶技能**
- **Playwright/Puppeteer集成**
  - 无头浏览器自动化
  - 截图和调试
  - 网络请求拦截
  - 性能监控

- **数据提取和清洗**
  - 正则表达式匹配
  - 自然语言处理（提取JD关键信息）
  - 实体识别（公司、职位、地点）
  - 数据去重和合并

### **2. 智能筛选算法技能**

#### **Sponsor检测算法**
- **H1B数据库集成**
  - USCIS公开数据解析
  - 公司Sponsor历史分析
  - 成功率预测模型

- **JD文本分析**
  - 关键词匹配（sponsor, h1b, visa, work authorization）
  - 语义分析（"不提供sponsor" vs "提供sponsor"）
  - 否定词检测（"not", "no", "does not"）

- **经验要求分析**
  - 年限提取（"3+ years", "5 years experience"）
  - 级别识别（"entry-level", "senior", "principal"）
  - 弹性判断（"or equivalent experience", "new grad welcome"）

#### **公司分类算法**
- **AI公司识别**

  - FAANG公司列表

## ⚠️ 技术风险和应对

### **高风险技能**
1. **LinkedIn反爬虫**
   - **风险**：账号被封，IP被禁
   - **应对**：模拟人类行为，添加延迟，使用代理

2. **ATS系统多样性**
   - **风险**：无法处理所有申请系统
   - **应对**：逐步扩展支持，用户反馈驱动

3. **数据隐私和安全**
   - **风险**：用户数据泄露
   - **应对**：本地存储优先，加密传输，明确权限

### **中等风险技能**
1. **Claude API成本控制**
   - **风险**：API调用费用超预期
   - **应对**：缓存结果，优化Prompt，使用配额

2. **维护成本**
   - **风险**：页面结构变化导致失效
   - **应对**：模块化选择器，自动测试，快速更新

### **低风险技能**
1. **UI/UX设计**
   - **风险**：用户体验不佳
   - **应对**：用户测试，迭代优化，A/B测试

2. **性能优化**
   - **风险**：扩展运行缓慢
   - **应对**：代码优化，懒加载，缓存策略
