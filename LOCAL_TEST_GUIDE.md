# 本地Chrome浏览器测试指南

## 🎯 测试目标
在您的本地Chrome浏览器中测试US Resume Personal Assistant扩展的所有功能。

## 📋 准备工作

### 1. 确保Chrome浏览器已安装
- **下载Chrome**：https://www.google.com/chrome/
- **版本要求**：Chrome 88+（支持Manifest V3）
- **验证安装**：打开Chrome，访问 `chrome://version/`

### 2. 准备测试环境
1. **克隆或下载项目**
   ```bash
   # 如果使用Git
   git clone https://github.com/sylviaaaye/linkedin-resume.git
   cd linkedin-resume
   
   # 或者直接下载ZIP
   # 访问：https://github.com/sylviaaaye/linkedin-resume/archive/refs/heads/main.zip
   ```

2. **项目结构验证**
   ```
   us-resume-personal-assistant/
   ├── manifest.json          # 扩展配置文件
   ├── popup/                 # Popup界面
   │   ├── index.html        # 主界面
   │   └── app.js            # Popup逻辑
   ├── src/                  # 核心代码
   │   ├── background/       # Service Worker
   │   ├── content/          # LinkedIn监控脚本
   │   └── popup/           # Popup源代码
   ├── resume-analyzer/      # 简历分析器
   └── docs/                # 技术文档
   ```

## 🔧 加载扩展到Chrome

### 步骤1：打开扩展管理页面
1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 按Enter键

### 步骤2：启用开发者模式
1. 在扩展页面右上角，找到"开发者模式"开关
2. 点击开关，启用开发者模式
   - 开关变为蓝色表示已启用
   - 会出现"加载已解压的扩展程序"按钮

### 步骤3：加载扩展
1. 点击"加载已解压的扩展程序"按钮
2. 在弹出的文件选择器中，导航到项目目录
3. 选择项目根目录（包含manifest.json的目录）
4. 点击"选择文件夹"

### 步骤4：验证加载成功
扩展加载后，您应该看到：
- ✅ 扩展出现在扩展列表中
- ✅ 扩展名称为"US Resume Personal Assistant"
- ✅ 版本显示为"1.0.0"
- ✅ 扩展ID显示（如：`abcdefghijklmnopqrstuvwxyzabcdef`）
- ✅ 权限显示正确：
  - `storage` - 本地存储权限
  - `activeTab` - 当前标签页权限
  - `https://*.linkedin.com/*` - LinkedIn网站权限
  - `https://api.anthropic.com/*` - Claude API权限

### 步骤5：固定扩展到工具栏
1. 点击扩展列表旁边的"拼图"图标
2. 找到"US Resume Personal Assistant"扩展
3. 点击"图钉"图标固定到工具栏
4. 验证扩展图标出现在Chrome工具栏

## 🐛 常见问题解决方案

### 问题1：扩展无法加载
**症状**：Chrome显示"无法加载扩展程序"
**解决方案**：
1. 检查manifest.json格式：确保是有效的JSON
2. 验证文件路径：所有引用的文件都存在
3. 检查权限：确保权限配置正确
4. 重启Chrome：有时需要重启浏览器

### 问题2：Popup无法打开
**症状**：点击扩展图标无反应
**解决方案**：
1. 检查manifest.json中的popup路径
2. 验证popup/index.html文件存在
3. 检查JavaScript控制台错误
4. 重新加载扩展

### 问题3：Service Worker不工作
**症状**：存储功能失效
**解决方案**：
1. 检查service-worker.js
